# Health Care API

* Java 17
* Spring Boot 3.1.5
* Dependency Management: Maven
* Database: H2
* Unit Tests: JUnit 5 & Mockito
* Swagger

### How to Build :hammer:

```shell
mvn clean install -DskipTests
```
